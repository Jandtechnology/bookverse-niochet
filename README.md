# 📚 BookVerse

**BookVerse** es una aplicación web desarrollada con **React + Vite** que simula un catálogo interactivo de libros. Los usuarios pueden explorar títulos, ver detalles y agregar libros al carrito de compras. El proyecto está orientado a prácticas de desarrollo front-end moderno con componentes reutilizables, hooks, y diseño limpio.

---

## 🚀 Funcionalidades principales

- ✅ Visualización de productos (libros)
- 🔍 Búsqueda en tiempo real
- 🛒 Carrito de compras
- 📘 Detalle de cada libro (modal)
- 🎨 Interfaz adaptable y moderna

---

## 🧑‍💻 Tecnologías utilizadas

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [SweetAlert2](https://sweetalert2.github.io/) para notificaciones
- [CSS](https://developer.mozilla.org/es/docs/Web/CSS) personalizado
- JavaScript moderno (ES6+)
